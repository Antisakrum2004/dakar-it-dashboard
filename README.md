# dakar-it-dashboard

Статичный HTML-дашборд для отдела ИТ Дакар/АтиЛаб. Один файл `index.html` (~4700 строк), чистый HTML/CSS/JS, без сборщиков.

## 🔧 Установка для разработки

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Antisakrum2004/dakar-it-dashboard.git
cd dakar-it-dashboard

# 2. Создать локальный конфиг из шаблона
cp config.example.js config.js

# 3. Открыть config.js в редакторе и подставить реальные значения вебхуков Bitrix24
#    (вебхуки выдаются администратором Bitrix24 — НЕ коммитить!)

# 4. Запустить локальный сервер
python3 -m http.server 8000
#    и открыть http://localhost:8000 в браузере
```

## 🔐 Безопасность

**Файл `config.js` содержит вебхуки Bitrix24 с токенами доступа.** Он добавлен в `.gitignore` и **НИКОГДА не коммитится** в репозиторий.

Все секреты разделены на:
- `config.example.js` / `.env.example` — шаблоны с плейсхолдерами (в репозитории)
- `config.js` / `.env` — реальные значения (только локально, в `.gitignore`)

Если вы случайно закоммитили секрет — см. раздел 8.4 в `docs/BRAIN.md`.
